--Foreach Loop

1) Concept in my own words:

A foreach loop is used to go through every element in an array.
It is easier than a for loop when we do not need the index.
The loop automatically takes one value at a time.
It makes the code shorter and cleaner.
Foreach is mostly used for reading values.
I learned that foreach is useful for printing arrays.
This concept helped me understand array iteration.

2) C# Syntax

int[] numbers = { 10, 20, 30 };

foreach (int number in numbers)
{
    Console.WriteLine(number);
}

3) Eureka Exercise

I understood foreach when I printed all values from an array.
It was easier than writing index numbers manually.

4) Common Mistake

I sometimes try to use foreach when I need the index.
In that case, a for loop is better.
