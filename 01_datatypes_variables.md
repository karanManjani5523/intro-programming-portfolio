--Data Types and Variables

 1) Concept in my own words:

Variables are used to store information in a program. 
A datatype defines what kind of value a variable can store, such as numbers, text, or true/false values. 
In C#, variables must be declared before they are used. 
For example, int stores whole numbers while string stores text. 
Variables make programs dynamic because values can change during execution. 
I learned that choosing the correct datatype is important to avoid errors. 
This concept is used in almost every C# program.
