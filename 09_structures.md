--Structures

1) Concept in my own words:

A structure is a custom datatype in C#.
It is used to group related variables together.
For example, a Student structure can store name, age, and grade.
Structures help organize data in a better way.
This concept helped me understand how custom datatypes work.

2) C# Syntax

struct Student
{
    public string name;
    public int age;
}

Student s1;
s1.name = "Karan";
s1.age = 23;

Console.WriteLine(s1.name);

3) Eureka Exercise

I understood structures when I created a Student example.
It helped me see how one variable can hold many related values.

4) Common Mistake

I sometimes forget to create a structure variable before using its fields.
