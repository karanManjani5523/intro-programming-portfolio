--For Loop vs While Loop

1) Concept in my own words:

For loops and while loops are both used to repeat code.
A for loop is best when we know how many times to repeat.
A while loop is best when we do not know the exact number of repetitions.
In a for loop, initialization, condition, and update are written in one line.
In a while loop, the update is written inside the loop body.
I learned that choosing the right loop makes the code easier to understand.
Both loops are important in programming.

2) C# Syntax

for (int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}

int x = 1;

while (x <= 5)
{
    Console.WriteLine(x);
    x++;
}

3) Eureka Exercise

I understood the difference when I printed numbers from 1 to 10.
For loop felt easier because I knew the number of repetitions.

4) Common Mistake

I sometimes forget to increase the variable in a while loop.
This can create an infinite loop.
