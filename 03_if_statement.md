--If Statement

1) Concept in my own words:

An if statement is used to make decisions in a program.
It checks whether a condition is true or false.
If the condition is true, the code inside the if block runs.
If the condition is false, the code is skipped.

2) C# Syntax

int age = 20;
if (age >= 18)
{
    Console.WriteLine("Adult");
}

3) Eureka Exercise

I understood if statements better when I practiced checking if a number is positive.
It helped me see how conditions control the program.

4) Common Mistake

I sometimes forget parentheses around the condition.
Example, if age >= 18 is wrong.
