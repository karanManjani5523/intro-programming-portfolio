--Functions

1) Concept in my own words:

Functions are blocks of code that perform a specific task.
They help avoid writing the same code again and again.
A function can take input using parameters.
A function can also return a value.
Functions make programs cleaner and easier to understand.
I learned that functions should be written outside Main but inside the class.
This concept is important because big programs are divided into smaller parts.

2) C# Syntax

static int Add(int a, int b)
{
    return a + b;
}

int result = Add(5, 3);
Console.WriteLine(result);

3) Eureka Exercise

I understood functions when I made an Add function.
I realized that I can call the same function many times with different values.

4) Common Mistake

I sometimes forget to return a value when the function return type is not void.
